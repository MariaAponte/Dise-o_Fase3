INICIO

Objetivo general: Suministrar información para que el usuario pueda profundizar y tener información complementaria sobre el uso de GIT y GITHUB para gestionar proyectos colaborativos de diseño web, Sistemas de control de versiones y Editores de código para la web. Objetivo específico 1: Brindar y facilitar información a los usuarios de cómo crear un repositorio remoto y clonar en local mediante GIT y GITHUB. Objetivo específico 2: Brindar y facilitar información a los usuarios de la Estructura básica página HTML (HTML, HEAD, BODY), así como de la Estructura básica de html5 (etiquetas, atributos y valores). Objetivo específico 3: Presentar información acerca de la navegación externa e interna (NAV - UL - OL - LI)
